# Advanced HTML/CSS Project

This project is a responsive website built using modern HTML and CSS practices.  
The project was developed as part of a frontend course and focuses on clean layout structure, responsive design, and SCSS architecture.

## 🚀 Technologies Used
- HTML5
- CSS3
- SCSS (variables, nesting)
- Gulp
- Responsive / Adaptive layout
- BEM methodology

## 📱 Features
- Fully responsive design (mobile, tablet, desktop)
- SCSS variables for reusable styles
- Optimized styles using Gulp build system
- Clean and structured layout

## 🛠 Build Tools
The project uses **Gulp** to automate development tasks:
- SCSS compilation
- CSS minification
- Live reload
